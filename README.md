
# CryptoPlace

**CryptoPlace** — современное приложение для отслеживания криптовалют, созданное с помощью React.js, Vite и Tailwind CSS.

[Живое демо →](https://cryptoplace-by-ivan.netlify.app/)

---

##  Технологии

- React.js  
- Vite  
- Tailwind CSS  
- CoinGecko API
